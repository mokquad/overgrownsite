Tutorial how to install Overgrown

  1. Step 1: Install 3uTools on your computer
  2. Step 2: Jailbreak your iDevice with Phoenix or other jailbreak tool
  3. Step 3: Download Overgrown dump on your computer
  4. Step 4: Install Apple File Conduit 2 from Cydia
  5. Step 5: Copy every folder from Overgrown dump
  6. Step 6: Go to 3uTools, go to File system (Jailbreak) and copy every folder you copied. Click Replace for every folder
  7. Step 7: Wait 20-40 minutes

After completing these steps, your iDevice with Overgrown should be ready to use!
